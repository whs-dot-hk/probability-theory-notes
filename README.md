# Markov's inequality
https://www.youtube.com/watch?v=vjYanZ1nsZg

For $a \gt 0$,

$`Y =
  \begin{cases}
    0 & \quad \text{if } X \lt a\\
    a & \quad \text{if } X \geq a
  \end{cases}
`$
## A first course in probability
$`I =
  \begin{cases}
    1 & \quad \text{if } X \geq a\\
    0 & \quad \text{otherwise}
  \end{cases}
`$

then

$I \leq \frac{X}{a}$
